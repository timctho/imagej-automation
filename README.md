# imagej-automation


## Setup
```
pip install -r requirements.txt
```

## From local files
```
python run.py --input_folder <path>
              --output_folder <path>
```


## From Google Drive files
```
python run.py --input_folder <google_drive_folder_id>
              --output_folder <path>
```
